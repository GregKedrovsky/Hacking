# NMap

## Quick Reference

| Description | Syntax |
| ----------- | ------ |
| Initial Scan | `nmap -Pn [target ip]` |
| Scan Open Ports | `nmap -Pn -sS -sV -sC -O -T4 -p [port,numbers] [target ip]` |
